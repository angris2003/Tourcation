# Tourcation

Tourcation is an award-winning hackathon project that helps tourists explore destinations and attractions based on travel interests. The site includes curated destination pages and early API integrations for location and weather-based travel information.

## My Contributions

- Helped build the travel recommendation concept and user flow

- Worked with APIs to match tourist preferences to relevant locations

- Helped develop and test the frontend experience

- Contributed to the hackathon presentation/demo

## Features

- Tourist-location recommendation flow

- Destination pages for major cities/landmarks

- API-based travel/location data

- Weather/location-related information

## Tech Stack

HTML, CSS, JavaScript, jQuery, location/travel APIs
